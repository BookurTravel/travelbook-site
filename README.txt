# TravelBook Website

This is a simple static website for TravelBook.

## Files
- index.html : The main webpage
- style.css : Styling for the site
- images/ : Folder to store images (banner.jpg, paris.jpg, bali.jpg)

## How to Run Locally
1. Open the folder `travelbook-site`.
2. Double-click on `index.html` to open in your browser.
3. The site will load locally.

## Contact Form
To make the contact form work:
1. Go to [Formspree.io](https://formspree.io).
2. Create a free form and copy the form ID.
3. Replace the `action` in `index.html` form with your form ID.
